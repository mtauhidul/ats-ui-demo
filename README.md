# ATS Web App

A modern Applicant Tracking System that helps you manage job postings, track candidates through hiring pipelines, and automate recruitment workflows.

---

## Features

- **Dashboard** - View analytics and metrics at a glance
- **Job Management** - Create and manage job postings with visual Kanban pipelines
- **Candidate Tracking** - Track candidates through customizable hiring stages with drag-and-drop
- **Application Review** - Review and approve incoming applications
- **Client Management** - Organize jobs by client companies
- **Team Collaboration** - Invite team members with role-based permissions (Admin, Recruiter, Hiring Manager, Viewer)
- **Email Automation** - Auto-collect and parse resumes from monitored email accounts
- **Interview Scheduling** - Schedule and track candidate interviews
- **Resume Parsing** - Automatic extraction of candidate information from resumes
- **Global Search** - Quick search across jobs, candidates, and clients (Cmd/Ctrl+K)
- **Custom Tags & Categories** - Organize candidates with custom labels
- **Email Templates** - Pre-built templates for candidate communication
- **Notifications** - Real-time updates on application and candidate activity
